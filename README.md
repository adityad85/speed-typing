# Speed-typing

A game built in react to type faster. Type the word in 5 sec or you loose.

Live Here

https://react-speed-type.netlify.com/

## Installation

OS X & Linux:

```sh
npm install or yarn
```

Windows:

```sh
npm install
```

## Development setup

Just clone the repo and run below command to start the game.

```sh
yarn start
or
npm run start
```
